# CW2

> Web-based User Management and Reporting System

CW2 is a PHP web application for managing users, viewing dashboards, and generating reports. It has a simple, modular structure and is easy to set up locally or on a web server.

---

## 📂 Project Structure

- **`admin/`** – Admin scripts and management pages  
- **`body.php`** – Main content page  
- **`dashboard.php`** – Dashboard for users and admins  
- **`edit.users.php`** – Edit user information page  
- **`includes/`** – PHP includes like database connections and functions  
- **`index.php`** – Entry point of the application  
- **`login.php`** – Login page  
- **`reports/`** – Reports generation and viewing  
- **`static/`** – CSS, JavaScript, and images  
- **`templates/`** – HTML templates  
- **`users/`** – User-related scripts  

---
